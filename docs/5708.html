它是一个测量javascript代码的时间、CPU、RAM和堆的性能的工具

安装

$ npm install --save js-meter

使用

const jm = require('js-meter')

const isPrint = true
const isKb = true       // or Mbconst m = new jm({isPrint, isKb})

for(var i=0; i<10000; i++){
    Math.random()
}

const meter = m.stop()// RAM        :  1080 kb
// HeapTotal  :  1024 kb
// HeapUsed   :  -7.2265625 kb
// External   :  0 kb
// CPU        :  3.344 ms
// Spend time :  1004 ms
