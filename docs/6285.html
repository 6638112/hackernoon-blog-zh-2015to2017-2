代币拍卖之后会发生什么

原文：https://medium.com/hackernoon/what-is-coming-up-after-the-token-sale-eefd1a1e7721?source=collection_archive---------16-----------------------#2017-09-11

Propy的代币销售将于9月15日结束，还有不到4天。对车队来说，这是激动人心的一个月！自我们推出代币销售以来，已售出超过1450万枚代币(1 pro = 1美元)，约有5000人参加了此次销售。我们收到了33，700多份就业援助和1000多份BTC。参加参观tokensale.propy.com

2016年，Propy开始了一段旅程，灵感来自于通过区块链为房地产资产带来自动化和流动性的想法。九个月后，该团队决定创建一个基于令牌的生态系统，并进行令牌销售。现在，代币销售接近尾声，我们正开始一项更大的冒险。

代币销售结束后，我们有新的挑战和目标要实现:

  1. 根据白皮书为令牌流通创建产品和基础架构。
  2. 创建能够自我维持的协议，并作为扩展分散式生态系统的基础
  3. 创造这样一种可以实现网络效应的秘密经济

我想分享在接下来的几个月里，为了实现这些目标，Propy将会做些什么。

有史以来第一次在区块链上购买房地产，使用PRO令牌

我们已经开始部署我们的路线图，并继续开发带有智能合约的交易平台。根据我们与乌克兰政府的谅解备忘录，我们将启动区块链上的第一笔房地产购买(随后是加利福尼亚州和佛蒙特州)。

这不仅是Propy的一大进步，也是区块链社区的一大进步。一项非虚拟资产将首次在区块链上转移，这是一项与现实世界相关的高价值资产。

在接下来的一周/几周内，通过Propy交易平台，买家将使用PRO令牌和加密支付(ETH)远程购买乌克兰的一处房产。

在这个试点项目中，乌克兰政府将正式接受和批准将由Propy的智能合同执行的交易。

由于这一试点，乌克兰政府正在确定通过智能合同进行实时远程交易所需的立法变革。最重要的是，由于这一试点，属性注册表正在完善。

职业代币的流通包括:

  • 由Propy的注册处对props进行收费
  • 将收集的pro分配到网络增长池(见下文)
  • 用户能够自行或通过物业后端从市场(交易所)购买专业代币

Propy最初将对第一笔交易收取100个PRO tokens。此外，还将引入一种定价机制，其中注册费用将不再是固定的，而是取决于资产的价值，并且与市场上的令牌价格成反比，还将包括内部“天然气”价格，即执行相关智能合同的成本。这种机制是必要的，以确保服务价格不会变得太低或太高。

几家交易所想交易职业代币。一旦令牌在未来几周/几个月内在交易所可用，团队将确定最佳交易所，并在用户没有足够令牌时，代表用户在后端为每次交易购买PRO令牌。或者，用户可以在任何交易所购买PRO令牌，并使用它们来购买Propy上的资产(将来可能会有其他平台使用Propy的分散注册中心)。

为物业的分散分类账引入dPOS

我们的工程师团队正在开发一个以某种统一格式上传标题的协议。我们打算鼓励将图书上传到产权登记处，并希望使用上传协议的上传者(经纪人、产权代理、政府)为此获得PRO tokens奖励。此外，我们的目标是确保他们上传的数据是合法的。目前，我们有一些基于利益主体共识授权证明概念的想法，如何解决这一问题:

  • 多个矿工(上传者)将添加标题的数据库到Propy的注册表中，只有匹配的标题(不同上传者的重复上传)将在注册表中得到奖励和验证。
  • 将有一个单独的角色，验证者将在通过一些外部来源进行验证后对标题进行投票。
  • 矿工(上传者)将在第一次完成契约(交易)后获得奖励，并添加新的所有权历史。

上传者作为矿工将获得职业代币奖励。

网络增长和捐赠池将如何实现网络效应

我们将继续发展网络增长计划，增加Propy用户数量和需要PRO token的交易数量。本周，我们已经完成了PRO token balance在我们应用程序中的集成，通过奖励Propy用户在应用程序中的各种操作，鼓励全球采用Propy平台。在第一个月，团队不会分发大量的代币(可能每月总共10万代币)作为奖励，因为这些程序将首先进行测试。

为了实现全球采用，作为网络发展计划的一部分，Propy在世界各地任命大使。大使应该在他们的地区胜任和执行工作。你现在可以注册了。截至今天，在美国几个州，英国，阿联酋，中国，俄罗斯，乌克兰和保加利亚都有大使。大使候选人也申请在南美、西班牙、韩国、泰国、坦桑尼亚和加纳代表Propy。

来自网络增长池的好处将分发给政府和其他大型机构行为者，以激励对平台的利用。示例:实体用属性的分散注册中心替换他们的注册中心。对于所有权的登记，PROs是必需的，因此我们将这些PROs授予实体，并且对于这些所有权的所有未来交易，PROs将是必需的。正如智能合同执行需要以太房地产购买执行将需要专业人士。因此，如果该实体注册了100k个标题，那么这些100k个标题的所有未来更改都将需要PRO tokens中的注册费。

因此，拥有数百万用户和数千本图书将吸引平台上越来越多的交易，被纳入网络增长池的PRO将进一步用于吸引新的数百万用户和数千本图书，然后进行交易，这将是网络效应的基础。我们的工程师目前正在开发一个模拟模型，以确定给每个用户多少令牌，以便有必要从市场上购买更多的令牌来执行房地产交易。

捐赠计划

Propy保留了15%的代币用于分配给教育和软件组织/基金会。这些代币将被冻结一年。它旨在增加区块链生态系统中区块链工程师的数量，以鼓励使用适当的协议，并支持整个区块链基础设施的改善。

我们将通过Reddit建立一个投票系统，任何人都可以选择和投票选出我们将支持的最有趣的项目。

拥有PROs的工程师将了解Propy的协议，Propy将有机会在全球范围内发现最优秀的人才，包括我们需要定制智能合同的国家。

实现网络效应是密码经济学的关键部分之一。创造这样一种新颖的商业模式，每一笔交易都会带来新的用户和几笔新的交易，这就是这种网络效应的机会。由产品用户和持有代币的区块链工程师组成的社区将不可避免地支持产品，从而保护代币的价值。
