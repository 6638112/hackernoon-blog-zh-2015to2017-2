在这篇博客中，我将向你展示如何选取一个现有的代码库示例，并将其缩减为你自己的、使用Node.js和 Cosmic JS 构建的漂亮的单页网站。这个 Stellar Cosmic JS App 是由 @ajlkn 在 HTML5 UP 前端搭建在light Node.js服务器上的单页网站。我将使用 Cosmic JS (一种API优先的CMS，使管理和构建网站和应用程序更快、更直观)来安装、编辑内容、部署和更新基于web的CMS 。

如果你还没有，那就从报名参加宇宙JS 开始吧。下面提供了有用的资源来简化您的开发操作。

1.创建新的存储桶

您的bucket的名称是您正在构建的网站、项目、客户端或web应用程序的名称。我将我的博客命名为“Node.js单页网站”,以保持示例博客的简洁。

2.安装Node.js单页“Stellar”App

一旦你注册并命名了你的桶，你将被提示从头开始或者“查看一些应用”。在这篇博客中，我简单地点击了右键“查看一些应用程序”，这样我就可以开始安装Node.js单页Stellar应用程序了。

应用程序安装选项

Cosmic JS 让你能够在编程语言之间进行过滤，比如 Node.js、PHP、React、AngularJS 等等。只需点击恒星图标下方的“安装”即可开始，或者访问宇宙JS应用页面。

3.部署到Web

我点击了“部署到Web”。然后，在部署web应用程序时，我可以开始编辑我的对象。您将收到一封电子邮件，确认您的web应用程序的部署。如果您在部署期间遇到任何问题，您可能会被转到 Cosmic JS故障排除页面。
