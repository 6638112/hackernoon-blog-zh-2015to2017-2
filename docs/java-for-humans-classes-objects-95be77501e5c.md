在你的生活中，你已经了解了许多种类的动物、人、建筑、运动、学校等等。在英语课上，我们把这些“人、地方或事物”称为名词。在 Java 中，我们可以把它们称为对象——你可以像看待英语中的名词一样看待 Java 中的对象。
