当我使用 Ansible (或者我尝试学习 Ansible )时，有时我想测试一些小的 j2 片段(在 Ansible 内的模板/when条件/其他中使用)。你可以 ipython (或者你最喜欢的repl)手动或者你可以使用(并贡献)我昨天构建的这个小工具。

我已经调用了 jinrepl (没有更好的名字)，它基本上将字符串赋值到j2.template中并呈现出来，没什么大不了的，但是有一些简洁的特性。

特色:

https://github.com/bechampion/jinrepl

安装:

pip install termcolor
pip install jinja2
git clone https://github.com/bechampion/jinrepl
cd jinrepl
python jinrepl.py

到目前为止，这几乎是您所需要的一切(使用当前的一组特性)

特征

  • 线金佳效果图
  • 过滤自动完成
  • 条件和语句自动完成

但是我最喜欢的是，嵌入式Vim！(哈哈just os.system("vim"))

它做得更多一些，但是你可以在github repo上找到关于它的所有内容。
