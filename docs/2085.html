开源项目

今年，我有幸参与了许多开源项目。在去年的不同时间，这些项目中的一些令人惊讶地获得了巨大的关注。😱😱😱😱

  • Laravel Hackathon Starter:Laravel web应用程序的全面的hackathon/MVP样板。
  • Laravel密码:防止你的用户在你的Laravel 5应用程序中输入愚蠢的密码。
  • 令人敬畏的开源应用:令人敬畏的开源web制作的精选列表&供开发者学习。
  • Laravel pay stack:Laravel 5 Lite包用于 Paystack 。
  • Mvrd :机动车登记信息搜索门户库。
  • jusi be PHP Lib:jusi be短信REST API服务的PHP客户端库【https://jusibe.com。还为它创建了一个 Laravel包和通知通道库。
  • Atom Aurelia片段 : Atom包:使用ES6语法的Aurelia片段。
  • Linda Ikeji CLI :供黑客、HR合伙人、HR经理和懒人阅读的CLI工具 Linda Ikeji 。💻 💁 🙋
  • Laravel Identify:Laravel 5包，用于检测用户的浏览器、设备、语言和操作系统。📦 📱
  • Laravel Ngsc : Laravel 5包，为尼日利亚的州和城市提供流畅的API。📦 🍀
  • Laravel报价:Laravel 5包，为您的Laravel应用程序提供从成功到编程到生活到DJKHALED的各种报价！🔑 🙏
  • Laravel Emoji: Laravel 5包，利用PHP 7 unicode特性的力量在你的Laravel应用程序中提供表情符号。😋
  • Vue twittershare: 一个Vue.js指令，用于将页面项目共享到twitter。
  • Ascii Faces : PHP包，在你的终端上展示漂亮酷炫的Ascii Faces，就像ᕙ༼ຈل͜ຈ༽ᕗ一样。
  • Laravel考试:这是一个Laravel考试项目，从初学者到专家都有由社区和我策划的问题。

你可以在我的 Github账户上查看我今年创作和工作的更多内容。

2017年呢？

我为2017年感到兴奋。简而言之，我已经给它贴上了卓越年的标签。正如叔叔 Victor Asemota 已经声明的，我已经激活了2017年的#beastMode。这些只是我来年统治世界计划的一部分:

  • 开源:我想做更多的JavaScript，写更多的开发工具和，让开发者的生活更加幸福。
  • 写作:我想多写一些技术文章，让自己的写作更上一层楼。我将努力写并出版3本书。这是一项非常艰巨的任务，但正如他们所说:如果你的梦想没有吓到你，那么你还没有开始。
  • 演讲:我想在大型会议上演讲，在一些聚会和会议上做更有力的&高娱乐性的演讲。
  • 我想去至少7个新国家旅行！
