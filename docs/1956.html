任何公司或组织的表示层对于成功都至关重要。幕后的工作通常不会以执行的方式报告，让知识专家看起来缺乏光彩，让销售团队看起来毫无准备。克服这一障碍的一个现成资源是格式化一个数字幻灯片，以便向客户演示，无论您是与他们在房间里，还是在介绍性会议后提交一个链接。

在这篇博客中，我将向你展示如何利用现有的代码库例子来构建你自己的数字幻灯片演示文稿 。 Slide Deck构建在 Impress.js 演示框架之上，运行在light Node.js服务器上。安装后，您可以在您的Cosmic JS bucket dashboard中配置所有布局、内容和动画。

对于这个示例应用程序，我将使用 Cosmic JS 。Cosmic JS是一个 API优先的CMS ，它使得管理和构建网站和应用程序更加快速和直观。通过将内容从代码中分离出来，Cosmic JS增强了开发人员的灵活性，同时确保内容编辑人员能够以最适合他们的方式规划和部署内容。我们将使用Cosmic JS来安装我们的示例应用程序，部署和更新来自基于云的内容管理平台的内容。

如果你还没有，那就从报名参加宇宙JS 开始吧。下面提供了有用的资源来简化您的开发操作。

1.创建新的存储桶

您的bucket的名称是您正在构建的网站、项目、客户端或web应用程序的名称。我将我的命名为“数字幻灯片”，以保持示例博客的简洁。

2.安装幻灯片应用程序

一旦你注册并命名了你的桶，你将被提示从头开始或者“查看一些应用”。对于这个博客，我简单地点击了右键“查看一些应用程序”，这样我就可以开始安装 Cosmic JS数字幻灯片了。

应用程序安装选项

Cosmic JS 让你能够在编程语言之间进行过滤，比如 Node.js 、 PHP 、 React 、 AngularJS 等等。

只需点击幻灯片组图标下方的“安装”即可开始，或者访问宇宙JS应用页面。

3.部署到Web
