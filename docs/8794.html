朋友和家人经常问我从哪里开始学习比特币和区块链。随着最近价格的上涨，这些要求也在同步放大。我想你们很多人都有过类似的经历。

我总是很乐意回应，因为人们在投资前接受教育是很重要的，但这些回应需要时间，而时间是稀缺的。我开始编制一份清单，这样我就可以有效地回应。

这个列表并不意味着全面，它是精选的。这意味着我遗漏了大量重要的内容。我这样做是因为新来者经常报告说，他们最初被这个领域的大量信息所淹没。我试着用一步一步的方式来展示它，让它更加平易近人。这是一个基础知识的配方，你可以在此基础上继续构建。

如果让我再做一次，我会这样做来开始新的生活…

> >未来十个小时

福克斯简介。以太坊经典叉或比特币现金叉就是例子。(1小时)

如果你想购买比特币或以太币，可以在像比特币基地或双子星这样的交易所开户。几个注意事项:我)这不是投资建议，只投资你能承受全部损失的东西。ii)立即设置双因素身份认证。 不要用短信作为认证方式 ，像Google Authenticator或者Authy这样的app更安全。顺便说一下，你真的应该在你的谷歌账户上设置2FA。iii)如果你的财富中有相当一部分用于投资，你会想看看硬件钱包，但这超出了本指南的范围。(1小时)

除了比特币，还有数以千计的其他区块链代币。用勇敢的新硬币的温柔的介绍潜入代币。然后，去 Smith + Crown (一家代币研究公司)，挑选几个最近的代币销售，阅读S+C分析。(2小时)

在撰写本文时，比特币基地交易三种加密资产:比特币(BTC)、以太币(ETH)和莱特币(LTC)。如果你希望扩展到这三个之外，你可以在像 Poloniex 或 Bittrex 这样的交易所开设一个账户来交易许多其他货币，但他们不接受像美元这样的法定货币，所以你必须先在像比特币基地这样的交易所购买BTC或瑞士联邦交易所，然后将其转移到你的另一个账户。此时你可能还会想下载 Blockfolio 来跟踪一切。(免责声明:(i) 这不是投资建议，(ii)我不认可这些交易所，只是用它们作为例子，这里是一个更长的按交易量排列的列表，这里是对其中几个的描述。(iii) 永远不要投资超过你愿意损失的总额 ) (1小时)
