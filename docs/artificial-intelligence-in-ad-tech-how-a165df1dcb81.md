# 广告技术中的人工智能。怎么会？

> 原文：<https://medium.com/hackernoon/artificial-intelligence-in-ad-tech-how-a165df1dcb81>

人工智能(AI)和机器[学习](https://hackernoon.com/tagged/learning) (ML)是最近创业界的两个关键术语。为什么不呢，它优化了我们做任何事情的效率。所以，广告技术怎么能落后。

在过去的几年里，广告技术产业面临着许多问题。像 Adblock 这样的技术的兴起只是增加了这个麻烦。除此之外，很少有广告技术公司关注用户体验和平台的长期增长，这只会增加麻烦。

但人工智能已经准备好改变这一切。

![](img/024be861fbaa09cbc182a07a45fa7a05.png)

这里是广告技术如何与人工智能合作，以获得更低的点击费价格，更高的点击率(CTR)和转化率，以及更大的投资回报率。

到目前为止，任何一家广告技术公司都对广告网络选择、网站广告投放和分析这三个关键决策感到困惑。

在这里，我们看到人工智能的使用如何帮助广告技术行业找到这些棘手问题的更好答案。

# 广告定位中的人工智能

开发人员不会在他们的电脑前讨价还价，以确定哪个广告位置将决定你网站的最佳收入，人工智能通过采用机器学习算法来研究历史数据，以找到目标用户群的相关广告，从而简化了同样的事情。人工智能算法使用**热图**来研究网站访问者的实际去向，从而将相关广告放在这些位置上，以关注用户。

热图以前从未在广告技术中使用过。人工智能肯定会改变这一点，同时详细研究每一张地图，为营销人员找到最佳广告位置。

> **人工智能还会记住网站的 UX，帮助出版商以最佳方式优化网站。**

# 广告网络选择中的人工智能

有许多广告网络向网站所有者提供不同种类的广告，他需要根据网站对广告进行分类。

这整个过程被称为**广告中介**，与网站的高收入密切相关。人工智能[技术](https://hackernoon.com/tagged/technology)在这方面有所帮助，既最大限度地减少了人力，又提高了效率，同时确保只有相关的广告才能到达最终用户。在基于人工智能的方法中，这些相关广告通过与网站相关的关键字来定义，广告将通过数据、事实&智能来放置，其中数据将是用户或网站的过去历史。UX，网站内容，地理&时间在这里将被算作事实。

采用人工智能进行广告优化的最佳之处在于，它通过采用面向数据的方法减少了人工努力，减少了人为错误，并且仅在数据对广告位置和广告网络选择这两个参数说“是”之后才投放广告。

> 当机器学习算法被用来寻找最佳广告用户匹配时，所有这些只会变得更好。

# 自动化分析

人们经常说，所有的广告技术公司都不太重视他们的数据分析！

对于任何广告技术来说，分析一直是一个问题，最终出版商都不太满意。然而，基于人工智能的方法将推动广告技术公司的报告和分析功能达到新的水平。

如果广告技术公司的分析开始影响内容和网站的用户界面/UX，那么即将到来的时代对于广告技术公司及其合作方(即出版商、广告技术平台，当然还有真正运营节目的人，即用户)来说是非常令人兴奋的。
