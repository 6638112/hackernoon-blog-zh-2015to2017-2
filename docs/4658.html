在Github页面上创建并发布一个Nuxt.js驱动的网站

原文：https://medium.com/hackernoon/create-and-publish-a-nuxt-powered-website-on-github-pages-3e922dfa7372?source=collection_archive---------6-----------------------#2017-06-13

如何在5分钟内从零开始建立和分享一个静态站点？

安装 vue-cli :

npm install -g vue-cli

创建项目(用您的项目名称替换my-project):

vue init nuxt/starter my-project
cd my-project
npm install

将nuxt.config.js更新为添加一个基础URL (用您的项目名称替换my-project):

router: { base: '/my-project/' },

安装推动方向:

npm install push-dir --save-dev

通过编辑package.json文件并在script部分的开头添加这一行，创建一个deploy命令发布到 Github页面:

"deploy": "push-dir --dir=dist --branch=gh-pages --cleanup",

连接到您的Github帐户并创建一个空的存储库。

然后提交您的代码并将其推送到您的存储库(用您的名字替换me，用您的项目名替换my-project):

git init
git add .
git commit -m "init"
git remote add origin https://github.com/me/my-project.git
git push -u origin master

生成网站并发布它:

npm run generate
npm run deploy

去你的站点https://me.github.io/my-project(用你的名字代替me，用你的项目名代替my-project)，然后享受结果。

您可以通过再次运行以下命令来发布您的更改:

npm run generate
npm run deploy
