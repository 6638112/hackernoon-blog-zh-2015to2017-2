# 理解承诺

简而言之，一个承诺:

“想象你是一个**小孩**。你妈妈答应你下周会给你买一部新手机。

你不知道下周你是否会拿到那部手机。你妈妈可以*真的给*你买一部全新的手机，或者*放你鸽子*如果她不高兴就没收手机。

那是一个**承诺**。承诺有三种状态。它们是:

1.  承诺是待定的:直到下周你才知道你是否会得到那部手机。
2.  承诺是**解决**:你妈妈真的给你买了一部全新的手机。
3.  承诺被拒绝:你不会得到一部新手机，因为你妈妈不高兴。
