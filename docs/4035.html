    1 != 2

认出来了！= Python 3.0中的不等式运算符是一个可怕的、令人痛苦的错误，FLUFL恢复了菱形运算符作为唯一的拼写。

13.InPynite？

>>> infinity = float('infinity')
>>> hash(infinity)
314159
>>> hash(float('-inf'))
-314159

散列是标识特定值的固定大小的整数。仔细一看，无穷大的散列是10⁵ x圆周率。有趣的是，hash(float('-inf '))在python3中产生-10⁵xπ，而在python2中则是-271828即— 10⁵ x e。
