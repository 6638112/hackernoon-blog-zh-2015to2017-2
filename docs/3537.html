在Ubuntu linux上安装Mysql

原文：https://medium.com/hackernoon/installing-mysql-on-ubuntu-linux-86e9b9db1a52?source=collection_archive---------17-----------------------#2017-04-06

SQL server 被认为是的最佳数据库。网上窗口。在我个人 意见中，与 Mono 类比的是 MySql 。

在这篇文章中，我们将看到如何设置它，它是Ubuntu 上的连接器。

我试着做一些Mysql来支持一个需要它并能在T21上运行的项目。NET在Ubuntu linux上。努力让所有东西都离开窗口。

在此过程中，软件中心有些发热:

最初，我尝试了APT 的工作台和 Mysql的单独安装。然而，由于我不是100%确定(和上面的月蚀)，我只是清除一切，并遵循确定的方式。

使用命令sudo apt-get install mysql-server安装mysql服务器

给了密码(忘记了 ):

使用sudo apt-get install mysql-client设置终端客户端，可以使用命令mysql -u root -p访问为根

和/或使用sudo apt-get install mysql-workbench从用户界面

设置Mysql连接器

点击此处下载连接器。选择.NET & Mono平台，启动下载过程。

提取下载的文件，并将其移动到您的首选目录。

目录的内部视图如下所示:

使用ls命令将cd移动到目录中，并确保MySql.Data.dll确实存在。我去了v4.5以确保我有所有的新补丁。

使用命令sudo gacutil /i MySql.Data.dll将其注册到全局汇编缓存(GAC)

要确认注册:

  • 使用命令cd /usr/lib/mono/gac转到目录/usr/lib/mono/gac
  • 使用命令ls列出其内容

如果注册成功，您会发现MySql.Data。
