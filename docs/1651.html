如何使用

受子弹杂志系统的启发，它遵循一个简单的语法:

  1. 用你目前的假设填写所有的占位词。
  2. 在左侧框中用点()标记最关键的行。
  3. 当你验证你的假设时，把圆点改成“X”。
  4. 重复步骤2-3，直到您的整个业务模型得到验证。
