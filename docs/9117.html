如何使用Vue和Cosmic JS建立作品集网站

在本教程中，我将向你展示如何使用 Vue 和 Cosmic JS 创建一个作品集网站。

我们将要建立的投资组合网站将有4页，分别是主页、技术、项目和开源。这个网站也将有3个对象类型有概况，技术和项目。

设置桶
要创建桶，登录 Cosmic JS 并点击添加新桶

选择从头开始

设置配置文件对象类型
点击添加对象类型菜单，将配置文件放入单数文本框

添加下面的元字段:

1.简介(纯文本输入)

2.图像(图像/文件)

3.关于(HTML文本区域)

4.脸书(纯文本输入)

5.Twitter(纯文本输入)

6.Github(纯文本输入)

7.页脚(纯文本输入)

设置技术对象类型
点击添加对象类型菜单，将技术放在单数文本框上

添加图像元字段(图像/文件)

设置项目对象类型

添加下面的元字段:

1.简短描述(纯文本输入)

2.图像(图像/文件)

3.技术(多对象关系)，选择技术对象类型

4.URL(纯文本输入)

5.开源(单选按钮)，添加两个选项有是和否

依赖关系

这个网站使用 Vue 来创建单页应用程序。对于路由器，使用 Vue路由器。它还使用ES6，因此需要 Babel 和 Webpack 用于模块捆绑。对于模板引擎，它使用 Jade 和 Stylus 作为CSS预处理器。

设置index.html

在这个文件中，我们包含了 Twitter Bootstrap 、 Font Awesome 、 Cosmic JS 配置和核心Javascript。第12行是安装 Vue app的地方。

Setup script.js

本站使用 Vue路由器进行路由，您可以在12–17号线上看到路由。以导入所需的组件。

设置App.vue

该文件包含路由器视图组件、页脚和全局样式。模板在第1–19行。App.vue所需的组件在第22–23行导入。第42–49行调用 Cosmic JS API，此时组件已安装并且样式在第54–77行。

设置MainMenu.vue

该文件用于网站的主菜单。该模板位于第1–11行，在该模板中，它使用来自 Vue Router 的router link组件来创建锚点，因此将自动包含活动类。样式在第13–30行。

设置Home.vue

该文件包含图像和关于文本。模板在第1–15行。第18行包括宇宙JS 。第33–42行在组件挂载时调用 Cosmic JS API。样式在第46–55行。

设置技术. vue

此文件显示技术对象类型的内容。模板在第1–15行。第18行包括宇宙JS 。第33–42行在组件挂载时调用 Cosmic JS API。样式在第46–55行。

设置Projects.vue和OpenSources.vue

这两个文件显示来自项目对象类型的数据，不同的是Project.vue在open_source metafield_key的metafield_value为Yes而OpenSources.vue为No时显示来自项目对象类型的数据。第18行包括宇宙JS 。第28–34行呈现技术数组，第38–48行调用 Cosmic JS API，元字段_key为open_source，元字段_value为No，对于开源页面，元字段_value为Yes。样式在第53–69行。

结论
这是使用 Vue 和 Cosmic JS 构建的作品集网站。希望你喜欢这个教程，希望能帮到你。
