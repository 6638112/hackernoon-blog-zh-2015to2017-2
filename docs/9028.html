(以及如何恢复你的故事)

流行的数字内容策展工具 Storify 已经宣布自2018年5月16日起不再提供: Storify停售 。

幸运的是，导出故事的程序已经计划好了。它是这样工作的:

1)访问您的Storify账户
https://storify.com/login

2)将鼠标放在一个故事上并激活View

3)进入主菜单...并激活Export

4)选择一种导出格式:HTML、XML或JSON

5)在浏览器的File菜单中选择Save as/Complete web page

6) 对每个故事重复步骤

作为一个建议，那些只由推特组成的故事，可以用推特的功能“ 时刻 ”手动重建。
