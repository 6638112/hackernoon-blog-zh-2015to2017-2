3。npm v5.0.0版发布

v5将npm向前推进了一大步，显著提高了它在几乎所有常见情况下的性能，修复了一堆由于架构造成的旧错误，并且总体上使它更加健壮和容错。

它带来了一些改变，让做monorepos的人、想要一致性/安全性保证的用户的生活变得更容易，并为git依赖项带来了semver支持。

5。N-API:用于本机模块的下一代Node.js APIs】

现有的本机模块是用C/C++编写的，直接依赖于V8和/或NAN APIs。这种依赖的结果是缺乏API/ABI的稳定性保证，需要为每个主要的Node.js版本更新或重新编译本机插件。

下一代，ABI稳定的Node.js API for native modules或N-API旨在解决这个问题，它为JavaScript虚拟机中的本机API提供了一个ABI稳定的抽象层。
