将Docker映像推送到EC2容器注册中心

在这篇短文中，我将向你展示如何使用 EC2容器注册 ( ECR )服务在 AWS 上设置 Docker注册，以及如何从亚马逊 ECR 推送&图片。

1 —配置存储库

首先登录到您的 AWS管理控制台 ，然后导航到 ECR :

点击开始:

为您的存储库选择一个名称，然后点击“下一个”:

此时，您将被带到存储库仪表板:

恭喜你！您已经创建了您的第一个 AWS Docker注册表。

为了从ECR推送docker图像或将docker图像拉入ECR，我们需要从 AWS CLI 获取docker登录信息:

| eval $(AWS ECR get-log in-region us-east-1)

发出以下命令，用存储库URI标记映像:

305929695733.dkr.ecr.us-east-1.amazonaws.com/nginx-custom码头标签

运行以下命令将其推送到存储库:

码头工人推305929695733.dkr.ecr.us-east-1.amazonaws.com/nginx-custom
