在这篇博客中，我将向你展示如何利用现有的代码库例子来构建你自己的漂亮的在线作品集网站 。这个漂亮的作品集网站是在服务器上使用Node.js，在前端使用Bootstrap构建的。它对手机完全敏感，包括一个联系表单和一个由 Photoswipe 支持的图库部分。虽然本博客中记录的应用程序是为摄影而设计的，但它是多用途的，因为它的所有内容都可以作为Cosmic JS中的内容对象进行编辑。

对于这个示例应用程序，我将使用 Cosmic JS 。Cosmic JS是一个 API优先的CMS ，它使得管理和构建网站和应用程序更加快速和直观。通过将内容从代码中分离出来，Cosmic JS增强了开发人员的灵活性，同时确保内容编辑人员能够以最适合他们的方式规划和部署内容。我们将使用Cosmic JS来安装我们的示例应用程序，部署和更新来自基于云的内容管理平台的内容。

如果你还没有，那就从报名参加宇宙JS 开始吧。下面提供了有用的资源来简化您的开发操作。

1.创建新的存储桶

您的bucket的名称是您正在构建的网站、项目、客户端或web应用程序的名称。为了保持示例博客的简洁，我将我的博客命名为“我的作品集”。

2.安装在线投资组合应用程序

一旦你注册并命名了你的桶，你将被提示从头开始或者“查看一些应用”。对于这个博客，我简单地点击了右键“查看一些应用程序”，这样我就可以开始安装 Cosmic JS在线作品集应用程序。

应用程序安装选项

Cosmic JS 让你能够在编程语言之间进行过滤，比如 Node.js 、 PHP 、 React 、 AngularJS 等等。
