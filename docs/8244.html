coimon——加密货币报价器CLI。

在您的控制台上查看加密货币的价格和变化。对于加密投资者和工程师来说，这是最好的CLI工具。

所有数据来自coinmarketcap.comAPI。

安装

$ npm install -g coinmon

使用

$ coinmon

厉害！不再有ui—只有原始CLI！非常适合我们这样的工程师！
