# 结论

我没有得到我所希望的“ [Elmlightment](https://medium.com/elmlightments) ”。它想与 React 和 Vue 竞争，但你用它不如用其他两个有效率。我没有像爱上 Vue 一样爱上 Elm。

几周前我就想分享我在 Elm 的经历，但是我觉得我做错了什么，我没有得到 Elm。但是当安德斯·霍姆勒写下他对埃尔姆的拒绝时，我知道我不是唯一有这种感觉的人。所以最后写了这篇文章。
