# 什么是个人品牌？

个人品牌是你身份的反映，它是现实、感知和你之间的交集。

![](img/1b1580efb0a5d13e633dc73219310935.png)

你的身份是关于你真正的**为什么**和你的**核心价值观**。

你需要对你的**信息**，你的**为什么**保持一致和一致。

> 你相信什么？

我的回答是:

> 我相信一次经历、一篇博客、一本书、一次演讲、一句话或一个词可以永远改变人生。
