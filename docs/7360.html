什么是创建组件应用？

复制和粘贴组件文件夹来创建一个新文件夹需要花费多少时间？还是在编辑器上创建自己的代码片段？

create-component-app是一个从终端()或编辑器 ) 生成不同类型的 React 组件的工具，帮助您节省大量时间，并防止复制粘贴造成的错误。

有了许多选项，您可以很容易地快速创建您的React组件！

  • 创建您的组件，从终端引导，有很多选择
  • 您可以在当前项目目录下创建一个配置文件
  • 您也可以从params 传递配置文件
  • 您可以使用自己的定制模板
  • 现在… 您可以使用社区中的模板🎉

为什么是中等文章？

因为我们注意到每个团队都有不同的方法来创建React组件，并且该工具不能帮助所有人。但是现在，社区可以分享他们自己的React组件模板！
通过这种方式，React社区中的每个人都可以使用不同的方式来创建组件。

如何通过3个步骤分享我的模板？

  • 放弃这个项目
  • 在模板文件夹中创建你的模板文件夹
  • 创建您的模板
