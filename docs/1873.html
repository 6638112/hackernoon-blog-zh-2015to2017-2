超越嬉戏的形式:乔尼·伊夫对无聊完美的追求

原文：https://medium.com/hackernoon/form-over-frolic-jony-ives-quest-for-boring-perfection-de3367638e37?source=collection_archive---------3-----------------------#2016-12-16

苹果仍然拥有市场上最好的工业设计，但它们不再有趣了

现在我正坐在一台27英寸的iMac前面。这是我拥有过的最好的电脑，5K显示屏，高色域，24g内存和512固态硬盘存储。它很漂亮，极简主义，就像他们自2007年改用铝以来发布的每一款iMac一样。

这也是我拥有过的可修改性最小的台式电脑。这种趋势也可以追溯到铝制iMac，像今天这样，只有RAM是用户可升级的。(自2012年以来，即使是更小的21英寸iMac也不再如此。)很难不去问:为什么在苹果所有的设计中，轻薄是优先考虑的？

你知道答案:乔尼·伊夫。现在很明显，他希望苹果生产的所有产品看起来尽可能接近纯玻璃，用最少的、不加修饰的金属框架，在功能允许的情况下，尽可能接近完整和对称。为了实现这个目标，Ive的团队完全愿意牺牲功能。女性Lightning端口比女性USB-C端口略薄，现在你知道为什么iPhone永远不会有USB-C端口了。抱歉。你很幸运，单端口MacBook的one port不是Lightning端口。(我有可靠的消息来源，那件事正在考虑中。)

这通常被描绘成是在继续受传统硬件束缚和迈向未来之间的选择。但如果你十年前使用的是Mac，你还记得Mac上的电源指示灯，无论是台式机还是笔记本电脑，在睡眠时都是慢慢脉动的，就好像在慢慢呼吸一样？或者笔记本电脑上的电池，无论是可更换的还是永久的，都可以让你在不打开或唤醒机器的情况下检查电量。或者，就在去年，电源插头改变颜色以显示充电状态的方式。所有这一切——连同闪亮的苹果标志和现在欢快的T2创业公司钟声——都消失了。

进步的代价，对吗？

几年前，肖恩·布兰克出版了一本关于“如何让好东西变得伟大”的书，名为 细节决定快乐 。这句话抓住了一个基本的悖论:我们希望我们的产品在日常使用中远离我们，然而那些将我们从仅仅是满意的顾客转变为粉丝的产品几乎没有以正确的方式引起人们对它们的注意。当我启动我的马自达时，它的显示屏会亮起“变焦变焦”几秒钟。这很愚蠢，但六年后它仍然让我微笑。

“引起人们注意的微小接触”与Ive的主导美学正好相反。他创造了你可以作为艺术品欣赏的美丽物体。你不禁惊叹于他的团队为了实现玻璃和金属的完美融合，为了制作似乎可以直接操纵数据的用户界面，为了让硬件在你使用时消失而付出的努力。在Ive的指导下，苹果推出了比其他任何大型消费电子公司都更接近科幻未来的作品。然而他的设计毫无创意。不会有哪个瞬间让你猝不及防，让你会心一笑。伊夫的作品从不渴望让你开心地咯咯笑。

软件也没有逃脱他对简朴的嗜好。软件UX的五年时代一直是扁平化、去除、无情地消灭skeuomorphism。“交通灯”窗口控件现在只是圆圈；漩涡状的理发师杆子进度条是简单的蓝色，带有细微的脉动；当我们把图标拉离码头时，我们甚至连一点烟都没有。我很惊讶iOS的图标在重新排列模式下还会晃动。我不确定说我们看到的是苹果追求轻薄的软件模拟是否公平，但我也不确定它不是。

我不会是第一个抱怨软件和UX质量明显下降的人，也不会是第一个质疑苹果放弃传统端口是否有点过于激进的人。但感觉这一直是交易的一部分，对吗？我们要拿走软驱，或者只给你这些奇怪的USB接口，或者把电池封起来，但是看看我们现在能把这个东西做得多酷！它不像市场上的其他任何东西。这是的乐趣。

这台iMac是我拥有过的最好的电脑，但是一点都不好玩。最奇怪的是我的机械键盘，这是苹果现在做梦都想不到的东西。(如此笨拙。)

是的，但是你一直在说Mac系列。未来在iOS！尽管我在过去的帖子中透露自己是Mac的支持者，但我认为这不仅是真的，而且总体来说是好的。我是科幻未来的粉丝，在这种未来中，我不会看到许多人坐在27英寸的显示器和键盘前满足他们的计算需求——即使显示器是全息的，键盘不是物理的。

但是男人，谈论“纯窗格玻璃”的理想，对不对？

苹果含蓄地提出的论点是，电脑——尤其是以iPad为代表的未来电脑——是电器。电器可以很漂亮，但不应该华而不实。他们应该集中精力。我们应该更喜欢厨房辅助立式搅拌机，而不是价格减半却能做两倍工作的塑料仿制品，因为它不能很好地完成这些事情和，而且它会在一年内坏掉。(此外，你可以做所有这些事情的厨房援助，无论如何；你只需要买一些加密狗。)

那都是真的。也许我最清楚。但如果你给我看一张桌子，上面放着一台iPad Pro、一台Surface Pro和一台Surface Book，并让我按照酷元素的顺序对它们进行排序，我很难把iPad放在最前面。微软并没有试图为小怪癖而高兴，这也无妨(“看起来你正试图给你的UX增添个性！我能帮忙吗？”)，但他们都是汗流浃背的小心思细节。苹果对制造过程的细节精益求精。这很好，但这不是一回事。

也许——仅仅是也许——一点点华而不实是可以的，即使它给产品增加了半毫米的深度，或者给用户界面添加了一点(喘息的)skeuomorphism，或者允许在阳极氧化铝外壳上使用稍微不那么克制、有品味的颜料。在90年代后期，给他们的电脑注入一点乐趣，甚至是怪异，帮助苹果从边缘拉了回来。可能是时候再注射一针了。

做厨房助手是个不错的目标，但你知道吗？他们出售将近36种颜色立式搅拌机。
