# 本指南的目标

Selenium 是一套经常用于测试应用程序的工具:它在网络浏览器中打开一个页面，检查其内容，特别注意错误。它还可以与稍后检查内容的页面进行交互。

阅读本指南后，您将能够:

*   在 [Node.js](https://hackernoon.com/tagged/nodejs) 中编写硒测试
*   在 Selenium 服务器上本地运行测试
*   使用 Buddy CI/CD 自动化测试流程
