轻松检查npm模块更新

如果你是一个纱用户，运行yarn upgrade-interactive是一个内置的命令，让你的生活超级容易。但是不，这篇文章是为那些铁杆npm用户而写的。

使用npm时，还有一个内置的工具来检查依赖性更新。npm outdated显示过期软件包列表。

但是有一个更好的工具！使用 npm-check-updates 来检查更新并更新您的package.json是显而易见的。

npm-check-updates是一个命令行工具，允许您将package.json或bower.json依赖项升级到最新版本，而不考虑现有的版本约束。

装置

npm install -g npm-check-updates

使用

$ ncunuxt           ^1.0.0  →  ^2.2.0
node-sass      ^4.9.2  →  ^4.9.4
vue-scrollto   ^2.11.0 →  ^2.13.0Run with -u to upgrade your package.json

那不是很容易吗？它显示哪些包有更新，并使用标志 -u 自动替换package.json中的版本。
