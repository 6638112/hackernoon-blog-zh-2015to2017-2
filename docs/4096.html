插曲

第1周:开始药剂

在第一周，您将学习Elixir编程语言语法及其工具的基础知识，并学习如何在进程间发送消息。我们以一个自我指导的练习结束，来建立一个你自己的独立过程。

  • 【001.1】仙丹入门 —语言入门，安装二郎和仙丹
  • 【001.2】基本的Elixir语法和概念 —学习语言的基础知识，以及如何运行脚本。
  • 【001.3】混合和模块 —使用混合工具并定义模块、函数和结构
  • 【001.4】流程和消息 —流程是并发的单位。我们将生成一些进程，并让它们进行对话。
  • 【001.5】Elixir Weekly Drip # 1和练习:String up caser“micro service”—更多阅读链接和一个免费练习，以构建一个过程，该过程使用发送的任何字符串的大写版本进行响应。

第二周:中级药剂

在第二周，你将学习如何使用ExUnit编写测试，语言本身的更多方面，以及如何管理状态。

  • 【002.1】中间药剂制备 —准备了解管理状态和测试。
  • 【002.2】具有状态的进程，以及代理 —管理具有进程和消息的状态，然后查看标准库中的代理模块，以获得更简单的方法来完成相同的事情。
  • 【002.3】‘for’Comprehensions——处理可枚举和集合的能力和优雅。
  • 【002.4】使用ExUnit进行测试 —深入了解ExUnit、测试驱动开发和doctests。
  • 【002.5】Elixir Weekly Drip # 2和练习:逆向波兰符号计算器 —文档链接、控制结构、可枚举数和构建逆向波兰符号计算器的练习。

第3周:检察官办公室

在第三周，您将了解OTP。这是标准库的一部分，旨在帮助您更好地对并发流程建模。它还提供了监督树，这是使用Elixir构建容错应用程序的基础。

  • 【003.1】准备学习动态口令 —逆向波兰符号计算器练习的解决方案，以及学习动态口令的预备读物。
  • 【003.2】GenServer和Supervisor —构建通用服务器和监督树
  • 【003.3】监督任务和代理 —使用监督人处理任务和代理中的细微差别。
  • 【003.4】genstate machine—用数字锁模拟一扇门，探索构建有限状态机。
  • 【003.5】Elixir Weekly Drip # 3和练习:受监督的RPN计算器和带式打印机 —有趣的链接和练习，使用GenServer和带式打印机构建RPN计算器。

第四周:埃克托

在第四周，我们介绍了Ecto，它可以帮助您与数据库进行交互。我们将开始为一个论坛构建一个数据模型，并以几个重要的查询结束。

  • 【004.1】准备学习Ecto —上周练习的解答和一些阅读材料，为我们学习Ecto做准备。
  • 【004.2】Ecto基础知识 —通过创建一个基本模式并对其进行查询来开始使用Ecto。
  • 【004.3】使用变更集的Ecto验证 —确保您的数据灵活地满足您的要求。
  • 【004.4】Ecto关联和更多查询 —探索如何在Ecto中对关联数据建模以及查询的更多细节。
  • 【004.5】药剂每周点滴#4和练习:更多查询 —一些与Ecto相关的链接和一个编写更详细查询的练习

第五周:凤凰城

在第五周也是最后一周的免费内容中，你将了解凤凰城，我们将从头开始建立火风暴论坛。Firestorm是一个开源的基于Phoenix的论坛引擎，旨在提供一个大型的、真实的Phoenix代码库，供人们学习。

  • 【005.1】准备了解凤凰 —解一道Ecto习题，准备了解凤凰。
  • 【005.2】开始使用Phoenix —创建一个新的应用程序并生成一些资源。
  • 【005.3】凤凰与前端 — HTML、CSS、JavaScript都是实实在在的东西。
  • 【005.4】Phoenix中的嵌套资源 —类别有线程，我们的URL应该反映这一点。
  • 【005.5】药剂每周点滴#5和练习:修复测试——修复上一集之后的测试，以及一些凤凰相关的链接。

第六周:开始大风暴

第六周是优质内容开始的地方，我们将从这里继续前进。本周我们从早午餐转向Webpack2，实现OAuth认证，研究Ecto。Multi，了解如何使用变更集更智能地处理一些事务，并处理视图和布局。

  • 【006.1】Webpack 2与Phoenix —在Phoenix应用程序中用web pack代替早午餐
  • [006.2]通过OAuth的认证 —使用Ueberauth向OAuth提供商认证用户。
  • 【006.3】向具有数据库事务的线程添加帖子 —使用Ecto.Multi处理数据库事务。
  • 【006.4】在Phoenix中使用Ecto关联 —更智能地修改和插入关联数据。
  • 【006.5】凤凰视图、模板和布局 —让Firestorm看起来更像它应该的样子。
