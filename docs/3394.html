为什么初次创业者听不进去

原文：https://medium.com/hackernoon/why-first-time-founders-cant-listen-5dc05ece4c4c?source=collection_archive---------11-----------------------#2017-03-30

作为一个曾经有过创业却没有按计划进行的人(咳 失败 咳)，我注意到的一件事是，当我们失败时，我们第一次创业的人总是说“我们希望人们以前告诉过我们这些”。我们会写满了已经流传多年的陈词滥调的创业建议的事后报告。专注一件事。销量是关键。解决人们实际存在的问题。除非你需要，否则不要筹钱。我们完全忘记了这样一个事实:在我们开始之前，许多人可能已经告诉了我们所有这些事情。我们只是没有接受。

当我想到为什么我们在开公司之前不听这个建议时，我想到了两件事:

  1. 我们被大量相互矛盾的建议淹没了，这些建议要么直接来自我们认识的人，要么来自阅读了太多关于创业公司的在线内容——几乎不可能知道哪些该听，哪些不该听。当一切都结束时，我们应该听取什么样的建议就变得更清楚了，因为现在我们知道了。
  2. 我们对我们的愿景如此专注和自信，以至于我们拒绝相信我们会陷入其他人都会陷入的陷阱。我们戴着有色眼镜，认为自己比任何人都聪明，比任何人都有决心，我们不会遭受和前人一样的失败。这种天真的无知使我们几乎不可能接受那些事后看来是如此荒谬明显，但在当时听起来却像是陈词滥调的建议。

我把这种无知比作人们谈论生孩子的方式。当你和有孩子的人交谈时，他们会告诉你很多关于它如何改变你的生活的事情，事情会变得不一样，处理睡眠不足有多难，等等。对于没有孩子的人来说，他们永远不会真正理解这一点，直到他们自己有了孩子。然后，他们会抛出陈词滥调。创始人也是如此。只有亲身经历过，你才能理解其中的挣扎。

优雅地说，我认为孩子的比喻也以一种更积极的方式适用于初创公司。溺爱孩子的父母总是会告诉你那种为人父母的感觉——从你的孩子身上获得的无条件的爱和快乐。这真的很神奇，但除非你亲身经历，否则完全无法理解。

我相信，经营一家初创公司也有类似的好处，即使这家公司最终失败了。兴奋带来的快感，追逐带来的肾上腺素。看到其他人使用和喜欢你做的东西，并向你展示他们辛苦赚来的钱有多少，这种快乐。最重要的是你获得的学习和知识——不仅仅是关于经营企业和制造产品，而是关于你自己和你想从生活中得到什么。

这个建议圈还有一个耐人寻味的地方，那就是我们创始人经常主动去寻求它，但之后却完全忽视了它。我们会给成功公司的创始人打电话，征求他们的反馈。我们会给我们崇拜的人发一封电子邮件，问他们是否可以请他们喝杯咖啡。然后当我们见面时，我们会向他们推销我们的想法，问一些范围广泛的问题，并要求非常笼统的反馈。这完全是在浪费大家的时间。

创始人为什么要这么做？

  1. 我们在寻求认可。我们希望成功的人告诉我们，我们有一个好主意，让我们感觉更好。
  2. 我们想要秘制酱。银弹。我们需要知道的一件虚构的事情会帮助我们成功。
  3. 通常，我们只是想要钱，但觉得有必要在需要建议的假象下掩饰谈话。投资者对此负有部分责任，因为这通常是他们唯一的出路。
  4. 我们害怕失败。如果我们要求一些具体的东西，他们有可能会告诉我们做错了，会失败。如果我们只是就一个想法征求反馈，如果他们拒绝了，我们可以把它一笔勾销，因为他们不理解这个想法— 他们就是不理解。
  5. 这是每个人告诉我们要做的。我坚信有一个创业行业——一个公司、事件、投资者、政府机构和其他组织的集合，它们积极地从创业文化中获利。你总会遇到其他人，或者和其他人聊天。你应该向其他人推销你的想法。一些你应该参加的狗屁创业竞赛或活动。你应该带上一些顾问。好的人会告诉你残酷的事实。其他人会把你留在长手指上，告诉你你想听的，以防有一天你真的做到了。
  6. 我们似乎有这样一种观念，即创业有一个既定的过程。当然，它会有所不同，但通常我们会有这样一个完美的梦，它会沿着这样的路线进行:
    have idea => find co-founder => seek out advice and mentors => build first version of product => raise money => launch => hire a team => raise more money => grow => sell for billions.
    我们寻求建议是因为我们认为我们必须这样做，这是过程的一部分。
  7. 它填满了我们的日历。我们经常试图让自己忙起来，这样我们会觉得自己更重要。遇见重要的人对我们来说很重要。

直到为时已晚，我们才意识到唯一重要的验证在于我们产品或服务的目标市场。与其担心如何走出去并获得客户的未来战略，我们应该只是走出去并获得客户。

作为一个即将从第一次创业者身份毕业、开始在第二家公司工作的人，我发现自己反思了很多我所学到的东西。最有价值的事情之一是，当我看到好的建议时，我学会了识别它。我学会了听谁的，不听谁的。更重要的是，我现在知道，最重要的是知道如何把好的建议放在上下文中。

经验丰富的创始人、成功的商人、投资者和其他人可以提供很多智慧。这一次，当我遇到这些人时，我会准备好一系列具体、相关的问题，帮助我取得成功或认识到自己的失败。因为现在我知道了。
