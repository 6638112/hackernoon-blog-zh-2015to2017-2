在这篇博客中，我将演示如何安装和部署一个使用AngularJS、 Node.js 和 Cosmic JS 构建的AngularJS电子商务应用程序。这个电子商务应用程序允许您处理来自客户的订单，并接受商品和服务的条纹付款。在你的 Cosmic JS 仪表盘或网站的管理区管理库存。

建立一个电子商务解决方案从未如此之快。只需安装和部署该应用程序，只需点击两下，您就可以将您的Stripe keys设置为环境变量，开始接受订单并使用您的新AngularJS电子商务应用程序处理交易。
