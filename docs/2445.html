当谈到 Jest 时，人们通常会将它与 React 联系在一起，这是因为脸书花了很多时间来确保Jest非常适合React，但它远不是而仅仅是一个用于 React 应用的测试框架。
事实上，Jest可以很好地与任何使用(或者可以被编译成) JavaScript 的库或框架一起工作。

在本系列中，我将使用一个库并展示我用来配置Jest并准备好使用它的过程。
我希望它能帮助人们更好地使用Jest和其他框架。

我将从 Vue.js 开始，因为它将允许我显示不同的配置选项，并且因为它有一个服务器端呈现器，这是一个很好的快照测试的候选。

虽然Vue已经有集成了，但我会假装不存在，从头开始尝试建立一个新的框架。

我相信这种方式会更有用，因为人们在试图建立自己的框架时可能会遇到类似的问题。
