4.翻滚！

确保你的比特币核心已经赶上了testnet链。获得一些测试网硬币，超过1 tbtc，并开始翻滚:

cd NTumbleBit.CLI
dotnet run -testnet

通过运行NTumbleBit。CLI，每次循环后HiddenWallet都会收到一些硬币。testnet服务器的命名为1 tbtc。需要24个街区才能将你的第一枚硬币放进你的隐藏钱包，之后每两个街区一枚硬币。
