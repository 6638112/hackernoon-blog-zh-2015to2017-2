我昨天看到了这篇文章，演示了如何在 Ubuntu 上建立一个 Kubernetes 集群。虽然文章是善意的，但我觉得有义务举手大喊:“等等，其实比那简单多了！”

无论您是Kubernetes新手，只是想开始运行，或者您是Kubernetes管理员，正在寻找一种更简单的方法来部署和管理集群，我将向您展示在Ubuntu上实现这一点的最快方法。

剧透警报

在全新安装的Ubuntu Xenial上，只需要两个(！)启动并运行完整Kubernetes集群的命令:

sudo snap install conjure-up --classic
conjure-up kubernetes

conjure-up命令将启动一个安装程序，引导您在自己选择的云上安装Kubernetes。你可以选择AWS，Azure，Google，Joyent，Oracle，Rackspace，还有…localhost！

部署到本地主机将会创建一个完全包含在主机上的Kubernetes集群，但是所有的集群组件都被整齐地分离到单独的 LXD 容器中。
