介绍

我决定为开发者写一本简单实用的个人品牌指南。我将谈谈作为一名Android开发者，我现在正在做些什么来创建和提高我的个人品牌。我希望你能在这篇文章中找到有用的东西。

什么是个人品牌？

个人品牌是你身份的反映，它是现实、感知和你之间的交集。

你的身份是关于你真正的为什么和你的核心价值观。

你需要对你的信息，你的为什么保持一致和一致。

你相信什么？

我的回答是:

我相信一次经历、一篇博客、一本书、一次演讲、一句话或一个词可以永远改变人生。

让我告诉你一个故事。

2011年，我在全职工作的同时接受了毕业的挑战。这很有趣，我通过创建一个博客来记录我的旅程。但是经过两年的努力学习和牺牲，我对完成我的硕士学位有些怀疑。我实际上有倦怠和精神抑郁。我试图寻找动力，重新站起来，继续我的挑战。我从我的虚拟导师那里找到了灵感(托尼罗宾斯、加里·维纳查克蒂姆·费里斯和埃里克·托马斯)。我创造了一个策略，我开始聪明地学习，而不是努力学习。我又回到了游戏中。终于在2017年我毕业了，我真的很开心。

作为开发商为什么需要个人品牌？

它们是拥有个人品牌的很多理由。我会给出对我来说真正重要的四个。

  1. 建立你的声誉
  2. 分享你的技能和激情
  3. 在地图上变得可见
  4. 与同行业的人交往

第一步:跟随你的导师

要做的第一件事是创建一个你钦佩的人的名单，你想联系他们。我称他们为导师，因为他们是自己工作中的佼佼者。如果你像我一样，你想通过走最好的路变得更好。

让我们列出世界上使用twitter的最好的android开发者。

这是我的清单:

推特上的@ewilly1/Droid_Mentors

twitter.com

第二步:定义你的品牌

第二步是选择你的主题。1到3个主题就足够开始了。你会花足够的时间专注于你的话题来建立声誉。我的主题是Android开发和 Firebase 。

第三步:精心制作你的个人资料

单幅照片

选择一张你美丽的照片或一幅你的插图。

两张封面照片

你可以放一些你在做有趣的事情时的照片(在工作中编码，在会议上发言……)

你可以放一些和你的博客相关的东西

你可以宣传你做过的事情(书、安卓应用……)

这是我做的

3-生物

你的简历必须是原创的。用这些元素描述你自己:

  • job(你是做什么的？)
  • 造诣
  • 生活方式或咒语
  • 业余爱好
  • ＃符号

我是这样做的:

高级Android工程师和公共演讲者。出生在非洲，生活在欧洲。努力工作，努力娱乐，努力休息#Firebase学习者#咖啡爱好者和#电视节目迷

第四步:每天参与

下一步是承诺参与twitter中的对话。这意味着每天发微博，大量转发，关注和回答。

示例:

  • 我的一条推文:
  • 我的一条转发
  • 我的一个回答
  • 我喜欢的一条推特

额外收获:我在twitter上的工作流。

我使用 feedly 阅读android相关的博客文章，然后在我的twitter账户上发布最好的文章。

如果你有一些在twitter上提升个人品牌的小技巧，你可以写在下面的评论里。

结论

我认为发展个人品牌是一个漫长的旅程，而我才刚刚开始。我会做实验，我有时会犯错误，我会在过程中学习。

这不是蓝图，这是我的过程。如果你还没有个人品牌，我邀请你以我为榜样创建一个。
