1-在heroku中设置您的信用卡信息

2-安装附加计划程序

在项目目录下安装插件调度程序，为这个特定的项目激活服务

$ heroku addons:create scheduler:standard

3-安装脚本

点击顶部绿色框的install add-ons

设置您想要运行的脚本

npm run postVideo
