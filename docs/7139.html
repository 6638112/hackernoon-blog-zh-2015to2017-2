应用机器学习

机器学习有很多活动的部分。在这篇文章中，我将通过帮助你编写工作代码来揭开ML的神秘面纱。我会和你讨论下一步该怎么做。

首先考虑在机器学习出现之前，软件是如何编写的。软件工程师给计算机一步一步的指令来解决一个特定的问题。

我们举一个银行业的例子。假设我们想写一个程序来预测贷款申请人是否有可能偿还贷款。我们可以编写一个程序来分析这个人的个人资料，设置关键变量的参数:

  • 信用评分
  • 贷款金额
  • 贷款类型
  • 成员资格期限

基于问题的复杂性，这些调整参数和编写指令的手动步骤可能会变得复杂甚至不可能。想象一下为物体识别编码的复杂性。

现在一台机器只要看着例子就能学会解决问题。

通过机器学习，工程师可以训练一个ML模型，从成千上万借款人的贷款数据中学习。ML模型可以随着时间的推移不断改进，响应更多的数据和新的趋势。例如，在2017年Equifax安全漏洞之后，Equifax的信用评分可能不如其他两家信用报告机构的评分有价值。如果这反映在实际贷款结果中，ML模型将调整其参数，降低Equifax信用评分的权重。有了足够的数据，ML模型将训练自己找到最佳参数。

这种技术叫做监督学习。我们将在下面的教程中使用它。(其他流行的技术有无监督学习和强化学习。)

房价预测教程

学习机器学习最快的方法是建立一个学习机，所以让我们建立自己的家庭价格预测器。假设每栋房子的基本价值为24万美元，每间卧室的价格增加了1.5万美元。(我们使用“k”来缩写1000，以保持数字较小。)

| Bedrooms | Home Price |
|----------|------------|
|     0    |    $240k   |
|     1    |    $255k   |
|     2    |    $270k   |
|     3    |    $285k   |
|     4    |    $300k   |

预测房价需要一个简单的线性模型(y = mx + b)。我们可以使用这个公式:

Price of house = ($15k * # of bedrooms) + $240k

现在让我们建立一个机器学习程序来做这件事。使用训练数据，我们希望我们的模型计算出m和 b的值，我们知道它们分别是15和240。

在代码中，我们设置了一些在培训中使用的基本占位符和变量。然后我们正在编写一个loss函数，它是通过从prediction值中减去y(给定值或地面真值)来计算的。然后，我们将那个loss值传递给我们的optimizer。随着每一次迭代，我们的optimizer将通过更新变量m和b的值来尝试尽可能接近y和prediction的值。
