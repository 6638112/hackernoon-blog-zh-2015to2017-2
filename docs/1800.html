如果你还没有，那就从注册参加宇宙JS 开始吧。下面提供了有用的资源来简化您的开发操作。

1.创建新的存储桶

您的bucket的名称是您正在构建的网站、项目、客户端或web应用程序的名称。

2.安装React通用博客应用程序

一旦你注册并命名了你的桶，你将被提示从头开始或者“查看一些应用”。对于这个博客，我简单地点击了右键“查看一些应用”，这样我就可以开始安装 React Universal博客了。

应用程序安装选项

Cosmic JS 让你能够在编程语言之间进行过滤，比如 Node.js、PHP、React、AngularJS 等等。只需点击React Universal博客图标下的“安装”即可开始，或者访问宇宙JS应用页面。

3.部署到Web

我点击了“部署到Web”。然后，在部署web应用程序时，我可以开始编辑我的全局对象、对象类型和页面。您将收到一封电子邮件，确认您的web应用程序的部署。如果您在部署过程中遇到任何问题，您可能会被转到 Cosmic JS故障排除页面。

确认部署位置和分支
