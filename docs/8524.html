在Ubuntu linux和Windows上设置Java

原文：https://medium.com/hackernoon/setting-up-java-on-ubuntu-linux-and-windows-f60c910d39e3?source=collection_archive---------9-----------------------#2017-12-05

在设置好 Git 、 node 和 Cordova 之后，需要完成 Java 设置，以便在 Ubuntu linux 上准备好Ionic。

在Ubuntu Linux上设置Java

输入命令sudo apt-get install openjdk-7-jdk。输入 root 密码，让系统执行它的任务。

输入apt-cache search jdk

设置“JAVA_HOME”和“PATH”环境变量:

export JAVA_HOME=/usr/lib/jvm/java-7-openjdkT3】

完成，现在只需通过检查安装的版本来验证安装:

javac -version

在Windows上安装Java

在这里打开从下载的设置文件

按照所示说明进行操作:

选择安装组件—对于需要特定配置的服务器，这可能是必需的

选择安装路径

如果你已经安装了 Java ，将会显示一个警告

现在，我们只需要设置环境变量。
