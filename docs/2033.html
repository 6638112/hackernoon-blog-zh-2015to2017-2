礼遇

如果我们第一次使用 Monodevelop 在Mono 上运行ASP.NET应用程序，我们可能会收到以下错误:

在某些情况下，问题是xsp4服务器的安装不正确。

使用以下方式安装:

sudo apt-get install mono-xsp4

这就解决了上面的错误。

可选:通过搜索xsp4从 Ubuntu软件中心获取

在xsp4服务器安装就绪后。该项目将于 http://127.0.0.1:8080/ 开始运行

也可能出现目录访问错误说Access to the path “/etc/mono/registry” is denied，这可以通过简单地使用mkdir创建文件夹来解决

sudo mkdir /etc/mono/registry

并使用chmod设置正确的权限

sudo chmod uog+rw /etc/mono/registry
