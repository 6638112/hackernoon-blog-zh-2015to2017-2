大家好，

昨天我确实设法从我的android手机上提供了我的web simple应用程序。通过遵循一些简单的步骤，你也可以做到，这简直太简单了！

首先，我必须说这个实验只在用 Golang 和Android版本编写的 Iris web框架下测试和运行。

步伐

打开您的android设备，按照以下步骤操作。

  1. 安装并打开 termux
  2. 按顺序执行以下命令
$ pkg install git
$ pkg install golang
$ export GOPATH=/data/data/com.termux/files/home/go
$ go get -u -v github.com/kataras/iris
$ cd /data/data/com.termux/files/home/go/src/github.com/kataras/iris/_examples/hello-world
$ go run main.go

3.打开您最喜欢的浏览器，导航到
http://localhost:8080
