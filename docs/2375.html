许多想要构建应用程序的开发人员不需要完整的引擎/库来完成它。与AngularJS类似，Vue.js不是一个“成熟的框架”，它旨在成为一个简单灵活的视图层，适用于数据模型简单的应用程序和网站，并与应用程序的表示层更加匹配。

在这篇博客中，我将向你展示如何利用现有的代码库例子来构建你自己的漂亮的 Vue.js / Nuxt.js网站。 Cosmic JS 有一个使用 Nuxt.js框架构建的Vue通用网站的例子。它是由一个月前宇宙JS 发布的 Next.js网站改编的塞巴斯蒂安肖邦。代码在客户机和服务器之间共享，使得开发变得轻而易举。添加 Cosmic JS驱动的内容，你就把你的网站提升到了Nuxt.js的水平。

在这个示例博客中，我将使用 Cosmic JS 。Cosmic JS是一个 API优先的CMS ，它使得管理和构建网站和应用程序更加快速和直观。

如果你还没有，那就从报名参加宇宙JS 开始吧。下面提供了有用的资源来简化您的开发操作。

2.安装Cosmic JS Nuxt.js网站

一旦你注册并命名了你的桶，你将被提示从头开始或者“查看一些应用”。对于这个博客，我简单地点击了右键“查看一些应用”，这样我就可以开始安装 Nuxt.js网站了。

应用程序安装选项

Cosmic JS 让你能够在编程语言之间进行过滤，比如 Node.js 、 PHP 、 React 、 AngularJS 等等。

只需点击 Nuxt.js网站图标下的“安装”即可开始，或者访问 Cosmic JS应用页面。

3.部署到Web

我点击了“部署到Web”。然后，我可以在部署web应用程序时编辑对象。您将收到一封电子邮件，确认您的web应用程序的部署。如果您在部署过程中遇到任何问题，您可能会被转到 Cosmic JS故障排除页面。
