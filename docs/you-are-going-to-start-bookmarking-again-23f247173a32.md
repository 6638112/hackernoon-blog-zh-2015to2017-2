## 适合你工作方式的组织

我们都喜欢保持自己的风格。篮子给你完全的控制权，并在你需要的时候帮你找到它们。在类别中保存项目。添加适合你工作风格的标签。调整标题。*它是你的购物篮，按照你喜欢的方式整理*。
