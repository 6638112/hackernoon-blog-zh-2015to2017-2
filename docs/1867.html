在这篇博客中，我将向您展示如何选取两个现有的代码库示例，并将其缩减到您自己的、使用React Native和Node.js构建的漂亮的跨平台博客中。这款应用程序是三合一的:Web、iOS和Android应用程序，所有内容都由 Cosmic JS 提供支持。前往 GitHub repo 查看iOS和Android应用共享相同的 React原生代码库，这将开发时间缩短了一半。web版本运行在一个轻量级的Node.js服务器上。当两种代码库都用于跨平台博客时，开发人员和内容编辑人员可以轻松地通过一次点击将内容更新到他们的web应用程序、iOS应用程序和Android应用程序。

我将使用 Cosmic JS (一种API优先的CMS，使管理和构建网站和应用程序更快、更直观)来安装、编辑内容、部署和更新基于web的CMS 。

如何使用React Native和Node.js构建跨平台博客

1.创建新的存储桶

您的bucket的名称是您正在构建的网站、项目、客户端或web应用程序的名称。我将我的命名为“跨平台”,以保持示例博客的简洁。

2.安装Cosmic JS跨平台博客

一旦你注册并命名了你的桶，你将被提示从头开始或者“查看一些应用”。对于这个博客，我简单地点击了右键“查看一些应用程序”，这样我就可以开始跨平台博客 的安装过程。

应用程序安装选项

Cosmic JS 让你能够在编程语言之间进行过滤，比如 Node.js、PHP、React、AngularJS 等等。只需点击跨平台博客图标下的“安装”即可开始，或者访问宇宙JS应用页面。

3.部署到Web

我点击了“部署到Web”。然后，在部署web应用程序时，我可以开始编辑我的对象。您将收到一封电子邮件，确认您的web应用程序的部署。如果您在部署过程中遇到任何问题，您可能会被转到 Cosmic JS故障排除页面。
