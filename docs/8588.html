TL；DR electron-toolkit是一个轻量级、小型、非常强大的开源应用程序，用于启动电子应用程序。应用程序本身运行在电子上(“耶递归！”).电子工具包可以在您现有的电子项目中从npm安装和启动。该UI将引导您使用一套工具来生成资产和工件，如图标、屏幕截图、安装程序、二进制文件——成功发布产品所需的一切。它可以帮助您发布应用程序，甚至生成完整的网站。这些工具可以节省几天、几周或几个月的工作。这是一个早期/预览版——希望更多的设置和工具将很快推出。

入门指南

电子工具包(ET)是npm包的一个全新类型的成员，它融入了命令行工作流，同时还提供了一个用户界面。

如果您有一个电子项目，您可以通过命令行轻松安装电子工具包:

npm install electron-toolkit --save-dev

ET重量极轻，体积很小。招数:它知道你要发布一个电子app。因此，它不会带来自己的电子二进制文件，而是会找到并重用您的应用程序正在使用的电子版本。这也意味着，如果你试图在一个项目之外独立启动它，它可能会以一种可怕的方式死去…但这没关系。只是被警告。

安装完成后，将启动脚本添加到package.json中，或者使用以下命令直接启动它:

"scripts": {
  "electron-toolkit": "electron ./node_modules/electron-toolkit" 
}

使用以下命令运行脚本:

npm run electron-toolkit

应用程序启动后，您应该会看到类似这样的内容。

电子工具包附带了一套工具，使你的应用程序的启动尽可能简单快捷。工具按类别分组，您会发现一些用于资产生成、安装程序创建、发布、网站创建等。

资产

以前，人们必须打开浏览器，进入一个网站，将他们公司的徽标上传到这些粗略的图标web服务之一，下载一个包含所有平台图标的zip(或病毒),并将其提取到正确的文件夹中。这个过程现在可以用两次点击来代替。

核标准情报中心

电子工具包有一个内置的图标生成器。你可以把你的logo拖放成svg，生成png包，ico for windows和icns for MAC——本地和离线的一切。ico和icns生成由伟大的png2icons模块提供支持，该模块实现所有低级位操作、重采样和图像缩放。Chromium中canvas元素的一个实例用于从可伸缩的矢量图形创建png。

截图和视频

资产类别包含另一个工具:屏幕截图

截屏工具使创建截屏和视频在营销活动中展示您的应用变得极其容易。你所要做的就是点击蓝色的大按钮。

电子工具包使用上下文信息来查找和启动正确的主应用程序。然后，它与另一个应用程序进程通信，以自动确定正确的应用程序窗口。接下来，它建立另一个应用程序的视频流作为实时预览。当主应用程序被其他窗口部分或全部覆盖时，预览也可以工作。你可以设置主应用程序的用户界面，然后再点击一下，电子工具包就会开始记录或远程保存截图到图库。

安装人员

配备图标后，我们可以继续下一个工具:安装配置器。电子工具包有助于配置电子建设者和创建多个目标的安装程序。

它将使用应用程序的package.json中的信息来生成一个表单，并用最重要的设置预填充表单。

安装程序配置器在获得焦点时并排显示每个设置的描述。描述和链接基于电子制造商的文档。有些字段已经有了基本的验证逻辑，并会产生警告，例如当mac被选为目标，但缺少应用类别时。

基于所选平台，高级选项卡将包含针对windows、mac或linux的其他目标特定设置。强大的电子生成器有无数的设置和选项。UI只显示那些对您当前配置有意义的设置，对它们进行分组并解释其效果。

点击“生成安装程序”，电子工具包将产生一个电子生成器的实例，并开始打包。你不必离开工具。它会将输出流直接转发到您的用户界面中，并修补您的project.json以反映您所做的更改:这允许您从CLI启动未来的构建过程(如果这是您的首选方法)。

请注意，目前您可以在一台机器上配置多个目标，但只能在各自的平台上构建二进制文件。

出版

现在我们已经有了生成的安装程序，我们准备好将我们的应用程序分发给数百万用户。可以在发布类别中找到完成这项任务的合适工具。此类别显示用于发布和托管生成的安装程序的在线服务和宿主列表。托管服务确保你的二进制文件可以在任何时间从网络上的任何地方下载。一些(理想情况下)扩展必要的资源，以便与您的用户群一起分发，并将二进制文件分发到边缘位置，以改善延迟，从而获得快速下载速度和满意的用户。

Launchfox

Launchfox是我们正在开发的一项新服务。它是所有电子相关工具的通道，这些工具不仅仅需要你的本地计算机。

使用Launchfox，您可以轻松地计划、跟踪和控制您的安装程序分发。像分阶段推出，下载计数和地区过滤器等功能将很快推出。您可以保留自己的DIY解决方案，例如两个GitHub回购(一个用于代码，一个用于发行和发布的公共回购)或Nuts-on-Heroku，并将其用作与现有解决方案集成的附加分发渠道。然而，我们的想法是为电子从头开始创建和设计一个解决方案，并使其成为最佳解决方案。

如果这听起来很有趣，请在 launchfox网站上留下你的电子邮件，并获得关于即将发布的通知。

网站(全球资讯网的主机站)

创建网站是一件耗费大量时间的事情。已经有一些服务可以帮助你加快这个过程，让你一起点击网站，比如Squarespace(即“作为基于服务的跟踪和数据收集服务的软件，作为内容管理系统集成网站生成器进行营销”——维基百科)。

但是因为你非常了解Web技术，所以你可能想建立自己的网站，并把它放在GitHub页面上。您将花几天时间将所有东西放在一起，设计样式，使其具有响应性，在不同的设备上测试，优化页面加载时间，缩小源代码，创建能够调试的源地图，捆绑脚本，捆绑资产，配置构建管道，考虑内容交付网络，针对不同的屏幕分辨率和设备类型缩放和优化图像，注册域，获得证书，设置https，创建自动部署脚本，到处注册帐户等等…

这需要很多时间。电子工具包可以帮助你完成这个过程。您可以使用package.json中的图标、截图、描述和您的产品信息，从该工具中引导一个外观精美的产品页面。

该网站已经实现了产品网站的最佳实践，并为访问者提供了一个干净和有吸引力的用户界面。您可以根据自己的喜好调整设置，重新设计风格，使用设备模板使您的屏幕截图突出并描述功能，添加评价，社交档案等。

您可以随时保存并继续工作。完成后，只需按“导出”即可将生成的页面以zip格式保存在您的计算机上。这项服务是由launchfox提供的，但是100%免费，没有数据存储在我们的服务器上。导出的webiste是您的，您可以按照自己的方式定制和使用它。
