React 16中的错误边界

原文：https://medium.com/hackernoon/error-boundaries-in-react-16-32fb8e185a3?source=collection_archive---------4-----------------------#2017-10-19

React 16来了它带来了很多激动人心的变化。让我兴奋的特性之一是更好的错误处理。以前，渲染期间的运行时错误可能会使React处于中断状态。

有了React 16，你可以使用错误边界，而不是每次出现错误就卸载整个应用。将错误边界想象成try-catch语句，但是是针对React组件的。
