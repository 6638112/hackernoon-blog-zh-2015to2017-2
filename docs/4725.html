您可以让Linus Torvalds提交到您的GitHub存储库

一般来说，你可以代表任何人做出承诺，并把它视为对名誉的要求。

诀窍是把你的git user.email改成与你想承诺的那个人的GitHub账户关联的同一封邮件。

尝试以下命令:

git config — global user.email “human@youlike.com”
