## 我的密集计划，以提高水平，成为一个更好的前端工程师

![](img/50450d136d21b923636b2c9230270b26.png)

Photo by [Arnold Exconde](https://unsplash.com/photos/0RWzJ17SKEs?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) on [Unsplash](https://unsplash.com/?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText)

我作为一名全栈开发人员已经从事专业编码工作 3 年多了，我特别喜欢前端工作。我是一名 bootcamp 毕业生，也是一名自学成才的开发人员，虽然我每天都在工作中学习很多东西，但我总是有一种冲动，想花时间完全掌握 JavaScript 和 T2 编程的深层基础，并成为一名更强的开发人员。

由于我为自己制定了一个强化计划，以提高和赢得技术编码面试，我决定与您分享。有时候你只需要看到计划展开就可以开始了，所以如果它能对任何人有用，我会很高兴。

我需要一个能让我真正**为代码面试**做好准备的程序，这样我就可以一边做全职工作一边跟着**，而且**不太贵**。**

对我来说，这个项目应该是密集的，这一点很重要:我发现在短时间的大规模行动中建立和保持动力比长时间涉猎更容易。我更像是一个短跑运动员，但当然如果你是马拉松型的，你可以很容易地将这个项目推广到更长的时间，以适应你的学习风格。

我还特意为每个课程选择了几个重叠的资源:从各种媒体(文本、视频)中学习，接触不同的概念呈现方式有助于更好地抓住内容，更不用说重复的好处了。
