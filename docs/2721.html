我已经和LiveScript合作了5年，发现这是有史以来最好的网络语言。当然，我在之前的项目中也尝试过其他语言，比如 Typescript 、 CoffeeScript 、 OCaml 、 Javascript 、 Haskell 、 C# 、 F# 。但最终还是决定使用LiveScript

  • JSON支持
  • 管道系统
  • 模式匹配
  • 部分应用
  • OCaml- like语法
  • 前奏库
  • 所有其他功能特性

如果你熟悉功能开发，你会爱上LiveScript。

大部分时间我都在和AngularJS 一起做前端工作，非常满意。

我周围所有使用 AngularJS 的朋友都开始使用 React 并提供了很好的反馈，但没有人使用 LiveScript 。他们更喜欢用面向对象的方式编写，或者懒得学习。

我决定也尝试一下 React ，但是有一个小小的条件，那就是我仍将使用LiveScript作为我的第一语言。

最后，我感到惊讶的是 LiveScript + React 非常友好，我甚至没有为了使用它而离开我的舒适区。
